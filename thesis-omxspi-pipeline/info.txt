Desktop Setup (Eikon)                                                                                                                                                     
                                                                                                                                                                          
- Copy the thesis-omxspi-pipeline folder to the school PC (zip/cloud/Git).                                                                                                
- Install Python 3.11+, open PowerShell, then:                                                                                                                            
    - cd thesis-omxspi-pipeline                                                                                                                                           
    - python -m venv .venv; .\.venv\Scripts\Activate.ps1                                                                                                                  
- Create a Desktop App Key in Eikon:
    - In the Eikon/Workspace search, open “App Key Generator”.
    - Click Create, copy the GUID (your App Key).
- Configure the project:
    - copy .env.example .env
    - Edit .env: set LSEG_SESSION_TYPE=desktop and LSEG_APP_KEY=<your app key>.
    - (Optional) lseg-data.config.json already defaults to "type": "desktop".
- Smoke test:
    - python -m src.run_pipeline --start 2019-01-01 --end 2020-12-31 --freq M --universe omxspi
    - Check data/processed/ for outputs.

If something fails

- “Desktop/Eikon not available”: ensure Eikon is open and logged in.
- “App Key not recognized”: regenerate via App Key Generator and update .env.
- “Not entitled” on fields: verify access in “Data Item Browser” and we can adjust field lists.
